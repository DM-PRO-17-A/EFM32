# EFM32

## Starter kit pins

### ADC tape sensor

PD0-PD7. ADC0 Channel 0-7.

### Motor

#### PWM

PE0: Timer3 channel 0 location 1.

PE1: Timer3 channel 1 location 1.

#### Direction

Use any of PE2, PE3, A12, A13, A14.

### Pynq Communication

PC0-PC7:

PC1 and PC2 next to USB-micro.

PC0 and PC3-PC7 on expansion header.
